百度搜索推广平台将在5月18日升级，升级后将取消多设备推广管理选项，您无需任何操作就可以实现推广结果在不同 屏幕上的展现，并且可以通过调整出价比例来体现您的投放意愿。

您现有的分设备推广计划（仅计算机、仅移动、全部设备）将会在升级时统一转为整合型推广计划， 即该计划下的关键词在移动设备投放的价格 = PC关键词出价*出价比例（{0}~10）

例如：“鲜花” 升级前出价为1元

如在投放设备为“仅计算机”的计划里，

  • 升级时，计算机设备上的出价不变，移动设备上的出价 = 关键词出价*{0} = 1*{0}

如在投放设备为“仅移动”的计划里，

  • 升级时，计算机设备上的出价=（升级前出价/出价比例上限）=1/10；
  • 移动设备上的出价 =（升级前出价/出价比例上限）*出价比例上限 = 0.1*10

其余计划，

  • 升级时，计算机设备上的出价不变，移动设备上的出价 = 关键词出价*1 = 1*1

建议您提前做好“仅计算机”与“仅移动”计划的合并和迁移，
